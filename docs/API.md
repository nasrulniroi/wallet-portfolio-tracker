# Wallet Portfolio Tracker API Reference

## Endpoints

### GET /api/data
Returns main dataset.

### GET /api/health
Health check endpoint.

### GET /api/stats
Returns aggregated statistics.

### Python API (port 8000)
- GET /health
- GET /api/data
- GET /api/stats

### Go Scanner (port 8080)
- GET /health
- GET /api/scan
